# 🚀 StepByDev

> Plateforme de reconversion digitale pour adultes en formation Full Stack Developer.

## 📌 Description

StepByDev est une application web pédagogique conçue pour accompagner
les adultes en reconversion professionnelle vers le métier de
développeur full stack.

Elle propose un parcours structuré, progressif et concret :
des leçons théoriques, des projets fil rouge, des sessions live
avec des formateurs, et un système de badges pour motiver la progression.

## 🎯 Objectifs

- Rendre la reconversion digitale accessible et structurée
- Accompagner chaque apprenant à son propre rythme
- Relier théorie et pratique via le projet fil rouge
- Valoriser la progression avec des badges et certifications

## 👥 Utilisateurs cibles

- Adultes en reconversion professionnelle
- Formateurs / Coachs techniques
- Administrateurs de la plateforme

## 🛠️ Stack technique

| Couche          | Technologie                     |
|-----------------|---------------------------------|
| Front-end       | HTML / CSS / JavaScript vanilla |
| Back-end        | Node.js + Express               |
| Base de données | SQLite (better-sqlite3)         |
| Auth            | JWT + bcrypt                    |
| Email           | Nodemailer + Mailpit (dev)      |
| Tests           | Jest                            |
| Déploiement     | GitHub Pages → VPS              |
